**Description**:

- Create react slim package with simple steps. 

1. npx react-chef sample (Project Name)
2. choose your project type (Slim)

   - Slim
   - Basic

3. choose your port number

   - 3000
   - 4000
   - 5000
   - 6000
   - 7000

4. run the app in development mode

- npm run dev

**Project Structure**

```
sample
├── node_modules
├── package.json
├── .babelrc
├──  webpack.config.js
└── src
    ├── components
       ├── ErrorHandler
          ├── ErrorHandler.js
          ├── index.js
    ├── static
       ├── images
       ├── index.html
    ├── App.js
    ├── index.js
```

### `npm run build`

Run the app in production mode

- npm run build
